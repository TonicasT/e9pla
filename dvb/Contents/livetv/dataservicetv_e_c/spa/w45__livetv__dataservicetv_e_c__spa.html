¿Qué es Data Service?
Esta función permite a los usuarios elegir entre MHEG1) (teletexto digital) y Teletexto si ambos existen simultáneamente.
Si solamente existe uno de ellos, se habilitará MHEG o Teletexto independientemente de la opción que haya seleccionado. 1) Este sistema cuenta con un grupo de expertos en codificación de información multimedia e hipermedia (MHEG, del inglés Multimedia and hypermedia information coding expert group) para ayudarle en la navegación por todas las opciones posibles de visualización.
  1. Pulse el botón Configuración del mando a distancia.
  2. Seleccione Icono Toda la configuración Canales Servicio de datos.
    (Podrá ver la Guía MHEG en Hong Kong.)
Las funciones de los servicios pueden estar limitadas cuando el TV no esté conectado a la red.
Cuando las funciones de Live Playback están activadas, este servicio se desactiva.
Cuando esté utilizando este servicio, las teclas de navegación y el botón Rueda (OK) pueden estar deshabilitados si el puntero del Magic Remote está activado en la pantalla. En este caso, pulse las teclas Izquierda / Derecha, Arriba / Abajo para desactivar el puntero. (Esto solo se aplica a los modelos compatibles con el mando a distancia Mágico.)
No se puede utilizar el servicio MHEG con el puntero del mando a distancia Mágico.
Esta función solo está disponible en algunos modelos.
