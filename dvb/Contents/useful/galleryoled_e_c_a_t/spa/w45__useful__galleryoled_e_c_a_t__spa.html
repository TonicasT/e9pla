Uso de GALERÍA
Puede disfrutar de una imagen de alta definición proporcionada para cada tema.
  1. Pulse el botón INICIO del mando a distancia.
  2. Ejecute la aplicación GALERÍA.

Aplicación de galería OLED

  1. Vea las imágenes integradas para cada tema.
  2. Es posible ver antes la información y la imagen del tema enfocado.
  3. Se puede eliminar un tema descargado o seleccionar el tema que desee reproducir.
  4. Salga de GALERÍA.
