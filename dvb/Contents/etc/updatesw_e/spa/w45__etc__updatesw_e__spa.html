Para actualizar el software
Utiliza Actualización de software para comprobar si existen actualizaciones y obtener la última versión.
También es posible comprobar manualmente las actualizaciones disponibles.
Verificar si hay actualizaciones
Configuración Icono Toda la configuración General Acerca de esta TV Buscar actualizaciones
Cuando se compruebe cuál es la versión actualizada más reciente, sólo podrá realizar actualizaciones si hay disponible algún archivo de actualización de una versión posterior.
La TV comprueba automáticamente la versión de software, pero puede realizar una comprobación manual si es necesario.
Actualización automática sin notificación
Configuración Icono Toda la configuración General Acerca de esta TV Permitir actualizaciones automáticas
Si está seleccionado Permitir actualizaciones automáticas, cuando una actualización esté disponible, la actualización se iniciará automáticamente sin solicitar su intervención.
Si Permitir actualizaciones automáticas no está seleccionado, cuando una actualización esté disponible, la actualización se iniciará automáticamente después de solicitar su intervención.
Los elementos configurables varían según el modelo.
Se puede obtener la versión más reciente a través de la señal de transmisión digital o de la conexión a Internet.
Si cambia de canal mientras se está descargando el software a través de una señal de transmisión digital, se interrumpe el proceso de descarga. La descarga se reanudará al regresar al canal inicial.
Si no se actualiza el software, es posible que determinadas funciones no funcionen correctamente.
En algunos países, Permitir actualizaciones automáticas no se muestra si no está conectado a Internet.
El dispositivo tendrá que descargar e instalar actualizaciones de software de forma ocasional para mantener un funcionamiento óptimo. Mientras se actualiza el software, en caso de no haber interacción por parte del usuario durante 4 horas, es posible que se apague la pantalla del dispositivo pero este no pasará al modo de espera automática hasta que finalice la instalación. Esto puede provocar un aumento del consumo de energía durante la actualización de software.
